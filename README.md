# modularity_metric
## Requirements
* gensim
* (optional) annoy
```
    pip3 install -r requirements.txt
```

Confirmed that it runs on 
* Python 3.6.5.
* gensim 3.4.0
* annoy 1.8.3

## Example Usage
```
    sh run_sample.sh
```
